# Команды
- apt (Advanced Package Tool) - Встроенный менеджер пакетов для установки, удаления ПО
- sudo - Выдача временных прав администратора
- ls - Показ скрытых папок git
- status - Проверка состояния рабочего каталога и индекса
- touch - быстрое создание пустых файлов
- rm - Удалить
- mkdir - Создание новой папки
- mv - Перенос данных
- rm - Удаление
- pwd - Путь у текущей папке
- add - добавление изменений из рабочей директории в индекс
	- add . - Добавить все изменения 
	- add <файл> - добавить конкретный файл
- git push - отправка локальных изменений в репозиторий
- git commit -  сохранение версии
- git init - инициализация нового локального репозитория
- git status - проверка рабочей директории и индекса Git
- mv - Переименовать файл
- .bashrc - домашняя папка линукс(ubunta)
- git switch - переключение между ветками
- git baranch - добавление новой ветки
- git merge - слияние веток
- git pull - обратна <git_push>, выгрузка с облака в локальную папку
	Устроена как:
		git fetch - скачка с сервера всех новых файлов в .git
		git merge - слияние новых файлов с текущими
Примерный код смены репозиториев
```
cd /mnt/c/Путь/К/Проектам/RPGCharacter
git status 
git add . 
git commit -m "Исправил логику генерации уникального ID" 
git push
```
# Флаги
- -a - Показать всё (all)
- -r - Рекурсивно
- -f - Без вопросов
- -i - Подтверждение y/n
- -u - Запомнить путь
- ~ - абсолютный ярлык домашней папки
- . - скрытый файл
